## рҹҡҖ Installation

Clone Repository

```bash
git clone <repository-url>
```

Navigate to Project Folder

```bash
cd portfolio
```

Install Backend Dependencies

```bash
cd backend
npm install
```

Run Backend Server

```bash
npm run dev
```

Server will start at:

```text
http://localhost:5000
```

Run Frontend

Open:

```text
frontend/index.html
```

using VS Code Live Server.
